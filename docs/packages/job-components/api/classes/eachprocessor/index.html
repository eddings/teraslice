Job Components: `EachProcessor`

Class: EachProcessor <T>

rejectRecord

▸ rejectRecord(input: any, err: Error): never | null

Inherited from OperationCore.rejectRecord

Defined in operations/core/operation-core.ts:96

Reject a record using the dead letter action

Based on OpConfig._dead_letter_action the transformation can be handled any of the following ways:

  • "throw": throw the original error
  • "log": log the error and the data
  • "none": skip the error entirely OR a string to specify the api to use as the dead letter queue
